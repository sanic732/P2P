Activate skill `p2p`. Launch **QUORUM** (sub-agent `p2p-quorum`) for the task: $ARGUMENTS

QUORUM will decide which specialists to spawn in parallel (TECTON / IRIS / FORGE / AXIOM / VECTOR / DATOS / ARCHITECTON),
apply weights by task type and Routing Memory bias (if data is available in `memory_bridge.md`),
synthesize consensus, and explicitly surface dissent.

Use for: Tier 2-3 tasks, multi-domain, high-stakes, production prompts.
